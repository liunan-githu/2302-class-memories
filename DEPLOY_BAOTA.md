# 宝塔面板部署指南

## 🐉 宝塔面板一键部署

### 第一步：准备 Node.js 环境

1. 在宝塔「软件商店」安装 Node.js 版本管理器
2. 安装 Node.js 16.x 或更高版本
3. 安装 PM2 管理器（宝塔软件商店搜索 PM2）

### 第二步：上传项目文件

#### 方式一：使用文件管理上传

1. 在宝塔「文件」中，进入 `/www/wwwroot/`
2. 创建文件夹：`2302-class-memories`
3. 将本地文件上传到此文件夹

#### 方式二：使用 Git（推荐）

```bash
# 在宝塔「终端」执行
cd /www/wwwroot/
git clone <你的仓库地址> 2302-class-memories
```

### 第三步：添加 Node 项目

1. 进入宝塔 → **Node项目** 选项卡
2. 点击 **添加项目**

**填写配置：**

| 配置项 | 内容 |
|--------|------|
| **项目名称** | 2302班回忆录 |
| **域名** | 你的域名 或 公网IP |
| **端口** | 3000 |
| **根目录** | `/www/wwwroot/2302-class-memories` |
| **启动文件** | `server.js` |
| **Node版本** | Node 16.x+ |

### 第四步：安装依赖

**方式一：在项目设置中安装**
1. 找到刚添加的项目 → 设置
2. 点击「模块」→ 安装依赖

**方式二：在终端执行**
```bash
cd /www/wwwroot/2302-class-memories
npm install
```

### 第五步：启动项目

1. 在 Node项目列表中，点击「启动」
2. 状态显示「运行中」就成功了！

### 第六步：配置反向代理（推荐，更安全）

#### 配置反向代理访问（80端口）

1. 在宝塔「网站」→「添加站点」
2. 填写：
   - 域名：你的域名或IP
   - 根目录：`/www/wwwroot/2302-class-memories`
   - PHP版本：纯静态

3. 添加站点后，进入站点设置 → 反向代理
4. 添加反向代理：
   - 代理名称：2302班回忆录
   - 目标URL：`http://127.0.0.1:3000`
   - 发送域名：`$host`

5. 现在可以直接用 `http://你的域名` 访问，不用带端口！

### 第七步：配置防火墙

#### 在宝塔「安全」中添加端口：

| 端口 | 说明 |
|------|------|
| 3000 | TCP 协议（如需要直接访问） |
| 80 | HTTP（反向代理用） |
| 443 | HTTPS（配置SSL证书后） |

### 重要注意事项

#### 1. uploads 目录权限
```bash
# 在宝塔终端执行
cd /www/wwwroot/2302-class-memories
mkdir -p uploads
chmod 755 uploads
```

#### 2. 修改默认密码
**部署后第一时间修改管理员密码！
- 默认账号：admin / admin123

#### 3. 配置 SSL 证书（可选但推荐）
在站点设置 → SSL → Let's Encrypt 申请免费证书

#### 4. 数据库备份
定期备份 `db.json` 文件

### 访问地址

| 访问方式 | 地址 |
|---------|------|
| 宝塔Node直接访问 | http://你的IP:3000 |
| 反向代理访问（推荐） | http://你的域名 |

### PM2 管理命令

在宝塔终端中可以用 PM2 管理项目：

```bash
# 查看项目状态
pm2 list

# 查看日志
pm2 logs class-memories

# 重启项目
pm2 restart class-memories

# 停止项目
pm2 stop class-memories
```
